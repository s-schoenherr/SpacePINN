# Data Layout

- `runs/` contains the provenance-preserving exported records: configurations, time series, summaries, source plots, and metadata.
- `figure1/` through `figure8/` contain convenience copies of the plot PDFs grouped by manuscript figure number.

The figure directories do not replace the exported records. They are intended for quick inspection and LaTeX inclusion; each directory includes a small `README.md` listing the source record for every copied PDF.
